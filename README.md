# KittenDOS

**Current Version:** 1.0a
**Author:** Kitten (Kitten Organization)
**Year:** 2026

KittenDOS is a simple educational x86 operating system inspired by MS-DOS.
Version 1.0a includes a bootloader, kernel, basic shell, and a minimal file system supporting up to 16 files.

## Features
- Bootloader displays: "Bootloader works."
- Kernel with prompt: `KittenDOS>`
- Commands:
  - `help` - shows available commands
  - `clear` - clears the screen
  - `ls` - lists files in the directory
  - `type <filename>` - displays file content
  - `write <filename>` - creates a new file
  - `format` - clears the file system
  - `exit` - halts the system
- File system: max 16 files, each up to 512 bytes
- Designed to run in x86 emulators

## Recommended Emulator

We recommend **Bochs** for the most accurate experience.

## Installation / Running

1. Download this repository.
2. Configure a disk image in **Bochs**.
3. Start the emulator and enjoy KittenDOS!
